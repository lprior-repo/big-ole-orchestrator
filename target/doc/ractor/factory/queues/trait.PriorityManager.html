Expand description

The PriorityManager is responsible for extracting the job priority from a given job’s key (TKey). Additionally in some scenarios some jobs may be non-discardable, i.e. can be enqueued regardless of the backpressure status of the factory. This is also responsible for determining if a job can be loadshed.
