Expand description

The settings to change for an update request to the factory at runtime.

Note: A value of Some(..) means that the internal value should be updated inside the factory’s state. For values which are originally optional to the factory, we use Option<Option<T>>, so if you want to UNSET the value, it would be Some(None).
